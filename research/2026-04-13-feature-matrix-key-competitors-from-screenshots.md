# Feature Matrix: Key Competitors from Screenshots

## Summary

This matrix normalizes the feature surfaces visible in user-provided competitor screenshots and includes a small number of search-backed corrections where later verification showed that a feature clearly exists on the competitor's public site.

It is intentionally conservative:

- linked `Yes` means the feature type is clearly visible in the reviewed UI surface or was later confirmed through a targeted site-specific search, and links to the nearest reliable source page or app screen
- `No` means it is not clearly visible in the screenshot

This is **not** a full product capability map.

It is a `publicly visible UI-surface matrix` based on the screenshots we reviewed.

## Important Caveat

This matrix reflects what is visible in the reviewed screenshots, not the full product truth.

Some competitors almost certainly have more features than shown here.

So this matrix is best used for:

- UI-surface comparison
- product-packaging comparison
- understanding what each company chooses to expose prominently
