# Social Media Guideline: Use Personal Statement, Not Generic CTA

## Summary

One useful marketing-writing rule:

- prefer `personal statement` framing like `I tried` over generic recommendation framing like `you can try`

This tends to sound:

- more human
- more credible
- less salesy
- less like generic AI-polished copy

## Core Idea

When possible, write from observed experience instead of telling the audience what they should do.

Stronger pattern:

- `I tried this`
- `we tested this`
- `I compared these tools`
- `I wanted to see if this actually works`

Weaker pattern:

- `you can try this`
- `give this a try`
- `check this out`

## Why It Works

`I tried` signals:

- first-hand experience
- proof instead of advice
- curiosity instead of instruction
- lower hype

It feels more like:

- a story
- an observation
- a real opinion

And less like:

- a promo line
- a generic CTA
- templated marketing language

## Tone Difference

`You can try this` usually feels:

- distant
- mildy promotional
- abstract

`I tried this` usually feels:

- concrete
- lived-in
- believable
- easier to trust

## Good Use Cases

This framing works especially well for:

- social posts
- founder voice posts
- product experiments
- tool comparisons
- reviews
- teardown content
- ad hooks that want curiosity instead of hard selling

## Example Directions

Stronger openings:

- `I tried building a game with this tool`
- `I tested 5 tools and this one surprised me`
- `I wanted to see if this actually works`
- `I spent 30 minutes trying this so you don't have to`
- `I thought this would be gimmicky, but...`

Weaker openings:

- `You can try this tool for game creation`
- `Give this a try if you want better results`
- `Check this out if you're building games`

## Practical Lesson

When the goal is attention and trust, start from:

- what happened
- what you noticed
- what surprised you
- what changed your mind

Instead of starting from:

- what the audience should do
- what they can click
- what they are being told to try

## Important Caution

Use personal-statement framing only when it is true, or when the brand voice clearly supports first-person experimentation.

If nobody actually tried it, `I tried` can feel fake very quickly.

## Working Heuristic

Prefer:

1. `I tried`
2. `we tested`
3. `I compared`
4. `I noticed`

Before defaulting to:

- `you can try`
- `give it a try`
- `check this out`

## Suggested Rule

Prefer `personal experiment framing` over `generic recommendation framing`.

When possible, write from:

- observed experience
- comparison
- curiosity
- surprise

Not from:

- weak CTA language
- distance
- generic advice voice
