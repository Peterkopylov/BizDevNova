# Consistency Tools Mentioned in Community Discussions

## Summary

This note is a running collection of tools, workflows, and approaches mentioned in community discussions about:

- character consistency
- face consistency
- body consistency
- identity persistence across images or videos

The goal is not to treat every mention as validation.

The goal is to build a market map of which tools repeatedly appear when users discuss real consistency problems.

## How to Use This Note

As more community threads appear, keep appending:

- source thread
- customer problem being discussed
- tools mentioned
- whether the tool is mentioned as a recommendation, workaround, or weak substitute

## What To Watch For

Over time, useful patterns to track include:

- tools that appear repeatedly
- whether users trust off-the-shelf tools or still rely on custom workflows
- whether discussion focuses more on face consistency or full-person consistency
- whether the market is converging on a standard workflow or staying fragmented
