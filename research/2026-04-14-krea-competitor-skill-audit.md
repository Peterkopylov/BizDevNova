# Krea Competitor Skill Audit

## Purpose

This note checks how the new competitor-skill architecture performs on `Krea` and compares the result with the existing note:

- [2026-04-10-krea-competitor-note.md](./2026-04-10-krea-competitor-note.md)

The goal is not to rewrite the Krea note fully, but to see:

- what the old note already captured well
- what the new skill structure improves
- what is still missing or ambiguous

## Existing Note: What It Already Does Well

The existing Krea note is already strong on the core commercial profile:

- good summary of Krea as a broad AI creative suite
- strong `Why It Matters`
- accurate `Key Selling Point`
- reasonable `ICP`
- useful `Observed Positioning`
- useful `Product Angle`
- strong `Highlighted Features in Communication`
- strong pricing capture
- useful `Official Site Updates / News Signal`
- user-scale claim capture
- review signal capture
- rights / commercial use capture
- control / consistency relevance
- GTM insight
- valuation / funding signal

In other words, the old note already performs well on:

- `competitor-commercial-signals`
- much of `competitor-feature-teardown`

## What the New Skill Architecture Catches Better

## 1. Headquarters / company-location signal

The old note does not have a dedicated `Headquarters / Company Location` section.

The new company-signals helper makes this explicit.

Best public signal found:

- `San Francisco, CA, United States`
- official LinkedIn company page also shows:
  - `781 Beach St, San Francisco, California 94109, US`

Source:

- [Krea LinkedIn company page](https://www.linkedin.com/company/krea-ai)

Why this is better:

- it gives us a standard place for company location
- it improves consistency across competitor notes

## 2. Public LinkedIn team signals

The old note has no dedicated section for visible people or hiring signals.

The new company-signals helper surfaces useful public team context:

- [Diego Rodriguez Prado](https://www.linkedin.com/in/asciidiego)
- [Víctor Perez](https://www.linkedin.com/in/iamvictorperez)

Why this is better:

- helps identify who to monitor
- gives us better founder / team context
- useful for outreach and ecosystem mapping

## 3. Official social presence

The old note does not explicitly track official social profiles.

The new skill architecture makes this a required surface.

For Krea, the strongest public monitoring surface is:

- `LinkedIn`

Why:

- highly active official update stream
- visible product announcements
- visible hiring signals
- visible follower count and company metadata

Source:

- [Krea LinkedIn company page](https://www.linkedin.com/company/krea-ai)

## 4. Better separation of signal types

The new skill structure is cleaner because it separates:

- company identity signals
- commercial signals
- feature/product signals

This improves note discipline.

For Krea, that matters because Krea has a lot going on at once:

- product breadth
- model-platform messaging
- pricing complexity
- public funding signal
- public team visibility

The old note captures a lot of this already, but the new structure makes the output easier to maintain.

## What the New Skill Does Not Magically Improve

The new skill architecture improves organization, but it does not automatically solve source ambiguity.

For Krea, the same hard parts still exist:

- some social metadata is stronger on LinkedIn than on the official site
- G2 is still low-volume and somewhat vendor-supplied
- Trustpilot is stronger but more complaint-heavy
- private valuation remains secondary-source territory

So the new system improves completeness and consistency, but not necessarily certainty.

## Best Combined Read on Krea

If we combine the old note with what the new skill structure adds, Krea now reads as:

- a broad creative AI suite
- a hybrid company with both own models and third-party model access
- based in `San Francisco`
- publicly active on `LinkedIn`
- visibly shipping fast
- visibly hiring across creative, growth, ML, and systems areas
- commercially strong in product breadth and pricing clarity
- externally mixed on trust/review quality

## Conclusion

The new skill architecture performs better on `Krea` mainly by adding missing structural fields rather than by overturning the old note.

### Main upgrade areas

- `Headquarters / Company Location`
- `Public LinkedIn Profiles / Team Signals`
- `Social Presence`

### Main conclusion

The old Krea note was already good.

The new skill architecture makes it:

- more complete
- more standardized
- easier to compare with future competitors

## Recommended Follow-Up

For high-priority competitors like Krea, FLORA, Higgsfield, Runway, and Canva, it is worth doing a light backfill so they all match the newer structure.
