# Yoodli Feature Surface From Screenshots

## Source

This note is based on the Yoodli screenshots shared in chat on `May 19, 2026`.

Context observed in the screenshots:

- scenario title: `Big TV Debate`
- scenario context: `Couple Shopping at Media Markt`

## High-Level Takeaway

The screenshots show that Yoodli is not just an AI roleplay tool.

It combines:

- multi-persona roleplay
- session playback
- transcript review
- coaching
- analytics
- speech and tone feedback
- in-line rewrite assistance

That makes it feel more like:

- a communication coaching and review workspace

Than:

- a simple one-shot simulation

## Visible Features

### 1. Scenario and Session Layer

Visible features:

- named session / scenario
- custom scenario context
- session timestamp
- session sharing
- left navigation with recordings/history

Observed example:

- `Big TV Debate`
- `Couple Shopping at Media Markt`

## 2. Multi-Persona / Multi-Party Roleplay

Visible signals:

- transcript contains separate turns from multiple participants
- scenario is explicitly framed around a couple shopping together
- this suggests support for multi-persona roleplay rather than only one learner versus one counterpart

This is important because it confirms a practical multi-party use case:

- pair-shopping retail scenario

## 3. Playback and Review

Visible features:

- session video / playback window
- timeline scrubber
- play / pause
- jump-back controls
- speed control
- volume control
- fullscreen

This suggests Yoodli treats the roleplay as something to:

- review
- revisit
- annotate

Not just complete once.

## 4. Transcript Layer

Visible features:

- transcript view below the playback area
- timestamped transcript entries
- copy transcript action
- comment-at-current-time input

This is a strong workflow signal because it supports:

- targeted review
- annotation
- post-session coaching

## 5. Coaching Layer

Visible features:

- `Coaching (6)` tab
- feedback blocks for strengths
- feedback blocks for growth areas
- follow-up questions section
- `Copy Feedback`
- `See more` expansion

This confirms that Yoodli’s product value is not only the roleplay itself, but also:

- structured post-session guidance

## 6. Strength and Growth Feedback

Observed coaching structure includes:

- positive reinforcement on what was done well
- growth recommendations tied to specifics
- concrete examples of what should improve

From the screenshot, examples include:

- understanding both customers' priorities
- needing more product specifics and evidence
- credibility risks from incorrect product information

This shows Yoodli is trying to coach:

- content quality
- recommendation quality
- persuasion quality

Not just delivery style.

## 7. Follow-Up Question Preparation

Visible feature:

- follow-up questions section with likely next questions the user should be prepared for

This is a strong feature because it expands the product from:

- session review

To:

- forward preparation

## 8. Pronunciation Coaching

Visible features:

- pronunciation section
- phoneme-specific explanation
- playback / `Play`
- targeted examples

Observed example:

- distinction between `/w/` and `/v/`

This suggests Yoodli has a fairly detailed speech-coaching layer, not only general delivery scoring.

## 9. Tone Analysis

Visible features:

- `Tone` section
- labeled tone tags

Observed labels:

- `Calm`
- `Helpful`
- `Under Pressure`
- `Confident`

This indicates a more explicit personality / delivery analysis layer than many simple roleplay tools show publicly.

## 10. Analytics Layer

Visible features:

- separate `Analytics` tab
- subcategories:
- `Word Choice`
- `Listening`
- `Delivery`

This suggests Yoodli separates:

- qualitative coaching

From:

- metric-style communication analysis

## 11. Word Choice Analytics

Visible metrics:

- weak words
- conciseness
- repetition
- filler words
- sentence openers

Examples shown in the screenshot:

- `5 weak words, 3%`
- `11% Excess`
- `8 repetition(s), 9%`
- `13 filler(s), 7%`
- sentence opener pattern like `"Yeah," 13%`

This is a strong analytics layer for communication-performance review.

## 12. In-Line Rewrite Assistance

Visible features from the transcript interface:

- `Rephrase concisely`
- `Simplify to key points`
- `Rewrite as a sonnet`
- `Convert to riddle`

The first two are highly relevant.

The latter two appear more playful, but they also imply:

- transcript-level text transformation tools

The practical product value is:

- helping users improve specific answers after the session

## 13. Iteration Loop

Visible features:

- `Practise` button
- transcript review
- coaching
- analytics

This confirms that the product loop is:

- roleplay
- review
- revise
- re-practice

That is a much stronger learning loop than a simple one-shot AI conversation.

## Strategic Interpretation

From these screenshots, Yoodli looks strongest in:

- communication coaching
- high-stakes roleplay review
- analytics on speech and wording
- multi-persona scenario support

It appears less like:

- a pure sales simulator

And more like:

- a communication performance platform with roleplay at the center
