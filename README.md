# BizDev Workspace

You can drop any business development material here:

- call notes
- partner ideas
- message threads
- market research
- ICPs / offers / hypotheses
- rough text drafts

I will turn them into structured `.md` files.

## Folder Structure

- `inbox/` - raw notes and quick drafts
- `companies/` - company and partner profiles
- `meetings/` - meeting and call notes
- `research/` - market, segment, and competitor research
- `insights/` - important market, product, and strategic observations
- `product-ideas/` - product and business ideas, hypotheses, and early opportunity notes
- `playbooks/` - processes, scripts, and internal playbooks
- `templates/` - markdown templates

## How We Can Work

You can simply send material in chat and, if useful, mention:

- where it should go
- what the file should be called
- whether you want a short summary

If you do not specify that, I will:

1. identify the material type
2. create a clear filename
3. place it in the right folder
4. structure it in markdown

## Default Note Structure

I will usually use a structure like:

- `# Title`
- `## Summary`
- `## Context`
- `## Key Facts`
- `## Next Steps`
- `## Risks / Open Questions`

I will adapt the format depending on the material type.
